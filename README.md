Chords.io
=======

Chords.io is a small jQuery plugin to display guitar chords using Raphaël SVG library.

Requirements
------------

Requires [Raphaël](http://raphaeljs.com/) and [jQuery](http://www.jquery.com).

Usage
-----

HTML

    <div data-code="xo221o" data-name="Am" />

JS

    $('div').chordsIO();
